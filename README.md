#  Movie Booking System

A full-stack Movie Booking Web Application built using:

- Node.js
- Express.js
- Oracle Database
- HTML, CSS, JavaScript

##  Features
- View available shows
- Select seats dynamically
- Create booking
- Auto total amount calculation (Trigger)
- Payment simulation
- Booking summary display

## Technologies Used
- Oracle SQL (Tables, Procedures, Triggers)
- Node.js Backend API
- Express Server
- Frontend UI (HTML, CSS, JS)
